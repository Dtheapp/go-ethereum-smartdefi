# Push to Fork - Commands Ready

## After Forking on GitHub

Once you've forked go-ethereum on GitHub, run these commands:

```bash
cd ../go-ethereum

# Add your fork as remote (replace YOUR_USERNAME)
git remote add fork https://github.com/YOUR_USERNAME/go-ethereum.git

# Push to your fork
git push fork master
```

## Or if you want to create a new branch:

```bash
git checkout -b smartdefi-precompile
git push fork smartdefi-precompile
```
